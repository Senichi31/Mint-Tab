# Mint-Tab

> **A sleek "New Tab" replacement designed to help you access and organise browser bookmarks with maximum efficiency.**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Version](https://img.shields.io/badge/version-1.1.0-mintgreen)

Mint-Tab transforms your default new tab page into a productive dashboard. It removes the clutter and focuses on what matters: your bookmarks, organised beautifully and accessed instantly.

---

## ✨ Key Features

*   **Foldable Navigation**: A dynamic sidebar that can be toggled to maximise your workspace.
*   **Smart Folder Access**: Automatically remembers your last opened folder for a seamless workflow.
*   **Modern UI/UX**: 
    *   Custom-designed iconography and typography.
    *   Support for **Grid Mode** with optimized bookmark icons.
    *   Fully scrollable sidebar for power users with extensive folder structures.
*   **AI-Assisted Design**: Optimised layouts for better focus and productivity.

## 🛠 Installation (Unpacked Extension)

Since Mint-Tab is currently in development and not yet on the Web Store, follow these steps to install it:

1.  **Download** this repository as a ZIP file and extract it, or clone it via git:
    This site was built using [GitHub Pages](https://pages.github.com/).
    ```bash
    git clone This site was built using [GitHub Pages](https://pages.github.com/).
    ```
3.  Open your browser (Chrome, Edge, or Brave) and navigate to the **Extensions** page:
    *   `chrome://extensions`, `edge://extensions` or `brave://extensions`
4.  Enable **"Developer mode"**.
5.  Click the **"Load unpacked"** button.
6.  Select the folder where you extracted/cloned Mint-Tab.
7.  Open a new tab and enjoy!
